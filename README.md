## About this app

This application is an admin platform to view products and carts. Using Next.js with Atomic design pattern.

## Getting Started for run in local

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
## Tech stack

- Next.js
- Redux
- Tailwind
- Axios
